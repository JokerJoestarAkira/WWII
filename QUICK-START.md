# 快速指南 | 三種玩法

## 方式1️⃣：本機遊玩（最簡單）

```
1. 下載 wwii-simulator.html
2. 雙擊開啟
3. 玩遊戲
```

✅ 進度自動存在你的電腦  
✅ 無需網絡、無需帳號  
✅ 完全隱私

-----

## 方式2️⃣：轉發進度給朋友

### 用分享代碼（推薦）

```
你：遊戲中點 📋 複製分享代碼
你：把代碼貼給朋友（一長串文字）
朋友：開啟模擬器
朋友：點 📌 貼上分享代碼
朋友：粘貼代碼 → 回車
✅ 朋友繼續你的故事
```

### 用 JSON 檔案

```
你：遊戲中點 📥 下載進度
你：用 WeChat/Email 傳檔案給朋友
朋友：開啟模擬器
朋友：點 📤 上傳進度
朋友：選擇你的 .json 檔案
✅ 朋友繼續你的故事
```

-----

## 方式3️⃣：在線分享版本（GitHub Pages）

### Windows 用戶

```
1. 雙擊 github-deploy.bat
2. 輸入 GitHub 用戶名
3. 輸入倉庫名稱（或按 Enter 用預設）
4. 按 Enter 完成登錄
5. 進入 GitHub Settings → Pages，選擇 main 分支
✅ 你的遊戲上線了！
連結：https://你的用戶名.github.io/wwii-simulator/
```

### macOS / Linux 用戶

```bash
chmod +x github-deploy.sh
./github-deploy.sh
```

然後進入 GitHub Settings → Pages，選擇 main 分支

✅ 你的遊戲上線了！  
連結：<https://你的用戶名.github.io/wwii-simulator/>

-----

## 進度管理

### 自動存檔

- ✅ 每回合自動存到本機瀏覽器
- ✅ 關閉遊戲後，下次自動恢復

### 手動操作

|按鈕      |功能            |
|--------|--------------|
|📥 下載進度  |存為 JSON 檔（備份用）|
|📤 上傳進度  |讀取 JSON 檔恢復進度 |
|📋 複製分享代碼|複製代碼轉發朋友      |
|📌 貼上分享代碼|粘貼代碼恢復朋友進度    |
|🔄 重新開始  |清空進度，重新建立角色   |

## 朋友常見問題

### Q：為什麼進度在不同電腦上不同步？

A：這是設計。每台電腦上的進度是獨立的。如果要分享進度給朋友，用「複製分享代碼」或「下載進度」。

### Q：為什麼我的進度遺失了？

A：可能是以下原因：

1. ❌ 清除了瀏覽器歷史/快取
1. ✅ 可以用 JSON 檔案恢復（先下載進度再清除）
1. ✅ 可以用分享代碼恢復

### Q：上線版本（GitHub Pages）和本機版有差別嗎？

A：沒有。功能完全相同。上線版只是方便朋友用連結遊玩，無需下載。

## 🎯 建議流程

**如果你只想自己玩：**

1. 下載 `wwii-simulator.html`
1. 雙擊開啟
1. 完

**如果你想分享給朋友：**

1. 下載 `wwii-simulator.html` 分享給朋友
1. 遊戲中用「複製分享代碼」或「下載進度」分享進度
1. 朋友導入進度繼續

**如果你想建立在線版本：**

1. 下載全部檔案到一個資料夾
1. 執行 `github-deploy.sh`（Mac/Linux）或 `github-deploy.bat`（Windows）
1. 在 GitHub Settings 啟用 Pages
1. 分享連結給朋友

-----

## 💡 提示

- 進度檔案（.json）可以重複使用，不會過期
- 分享代碼也可以保存，多次貼上使用
- GitHub Pages 版本更新後，朋友重新整理頁面自動使用新版本
- 所有進度都是本機儲存，沒有雲端同步
